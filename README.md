# Royal Gold Restaurant Website

## Project Overview

Royal Gold Restaurant is a modern, responsive restaurant website developed using HTML, CSS, and JavaScript.

The website provides information about the restaurant, menu items, pricing, and table booking facilities through a user-friendly interface.

---

## Features

- Responsive Design (Mobile, Tablet, Desktop)
- Modern Dark Theme UI
- Light/Dark Mode Toggle
- Interactive Navigation Bar
- Restaurant Information Page
- Food Menu with Images and Prices
- Table Booking Form
- Form Validation using JavaScript
- Attractive Card Layout
- Mobile-Friendly Design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Pages

### Home Page
Displays restaurant introduction, featured dishes, customer reviews, and services.

### About Page
Provides information about the restaurant and its mission.

### Menu Page
Displays food items with images and prices.

### Contact Page
Allows customers to book a table through a validated booking form.
